Hi! I'm excited about software engineering, data engineering, and functional programming.

I'm a software engineer with over a year of experience in languages including Python, Java, and Haskell. Most of my Python projects are Machine Learning related.

I know object oriented programming, functional programming, and databases. I can spin up a REST api or microservice on command. I know predictive modeling, machine learning, data visualization, and data analytics.

At every subproblem, I seek opportunities to write small, pure functions. Compositionality is paramount. I lint and typecheck my code as much as possible, because I've experienced strong ROI on code quality and bug-finding. I aspire to be thoughtful, opinionated, and consistent every time I have to decide between functions or classes, between recursive or iterative, and between stateful or stateless.

I'm happy to do any backend work or devops.

When I'm not writing Python, I'm writing Haskell or Coq. In Coq, I've automated parts of real & complex analysis.

My Triplebyte certification is in Machine Learning Engineering and I'd be thrilled to implement, deploy, and maintain your models!
